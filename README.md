# 🕸️ Code Weaver AI
A powerful, 24/7 Multimodal Discord Bot that handles Coding, Art, and Music—all without expensive API keys.

## ✨ Features
- **🤖 AI Chat:** Advanced coding help and conversation powered by `g4f`.
- **🎨 Image Gen:** High-quality AI art generation via `Hercai`.
- **🎥 Motion Clips:** Generate short animated GIFs from text prompts.
- **🧹 Pro Moderator:** A `/clear` command that can wipe up to 1000 messages.
- **📻 24/7 Radio:** Built-in Lofi radio that stays in voice channels around the clock.

## 🛠️ Setup
1. Clone this repository.
2. Install dependencies: `pip install discord.py[voice] g4f hercai python-dotenv`.
3. Create a `.env` file and add your `DISCORD_TOKEN`.
4. Run `python bot.py`.

## 🚀 Deployment
Designed to be hosted on **Railway** or **Render** for 24/7 uptime.
